# Flo Language Samples

This directory contains sample programs demonstrating Flo language features.

## Running Samples

Use the `flo` CLI to run any sample:

```bash
flo run samples/demo.flo
```

## Available Samples

### demo.flo
Comprehensive demonstration of all Flo language features including:
- Basic values (integers, floats, strings, booleans)
- Arithmetic operations
- Functions (named and anonymous)
- Control flow (if/elif/else, match)
- Loops (for, while)
- Collections (lists, maps)
- Error handling (attempt/rescue)
- Async/concurrency (strand, await)
- Mutable and immutable variables
- Constants

## Creating Your Own Programs

1. Create a new `.flo` file
2. Write your Flo code following the syntax in the spec
3. Run it with `flo run your-program.flo`

See the [Language Specification](../FLO_SPEC.md) for complete syntax reference.
